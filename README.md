# Seller App

> The best Seller app of world, by the best team.

## Getting Started

Install dependencies:

```shell
yarn
```

Start the dev server:

```shell
yarn start
```

Make your changes. 👩🏼‍💻👨🏻‍💻

Commit your feature:

```shell
yarn commit
```

Ship it!🚀
